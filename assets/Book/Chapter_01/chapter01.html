Chapter 01

1. Two Types of HTML Elements

HTML elements can be block-level (start on a new line) or inline (flow in line with text).

<p>This is a block element by Fahim.</p>
<span>This is inline.</span>

This is a block element by Fahim.

This is inline.

2. HTML Document Structure

The standard structure of an HTML document includes doctype, html, head, and body tags.

<!DOCTYPE html>
<html>
  <head>
    <title>Fahim's Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>

Hello, world!

3. Your First Webpage

Here's an example of a simple webpage written by Fahim:

4. Character Encoding

Character encoding ensures text displays correctly. UTF-8 is widely used.

<meta charset="UTF-8">

UTF-8 allows correct display of symbols like © ★ ✓

5. Element Attributes

Attributes provide extra info about elements, like ID, class, or href.

<a href="https://fahim.com" target="_blank">Visit Fahim's Site</a>
Visit Fahim's Site

6. Favorite Icon

The favicon appears in the browser tab. Place a favicon.ico in your site root.

<link rel="icon" href="favicon.ico" type="image/x-icon">
Look at your browser tab — that's where the favicon shows!

7. Comments

Use comments to leave notes that the browser ignores.

<!-- This is a comment by Fahim -->
No visible output — comments are not shown in the browser.

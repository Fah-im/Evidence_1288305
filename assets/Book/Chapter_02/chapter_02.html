Chapter 02

1. Headings

Headings from <h1> to <h6> define different levels of titles.

<h2>Welcome by Fahim</h2>

Welcome by Fahim

2. Paragraphs

Paragraphs are defined with the <p> tag.

<p>Fahim writes a clean paragraph.</p>

Fahim writes a clean paragraph.

3. Hyperlinks

Hyperlinks are created using the <a> tag.

<a href="https://example.com">Visit Fahim's Link</a>
Visit Fahim's Link

3.1 Internal Anchors

Link to a section of the same page using an anchor.

<a href="#images">Jump to Images</a>
Jump to Images

4. Images

Display images using the <img> tag.

<img src="./pexels-amir-1183434.jpg" alt="cat's Image">

4.1 Image Maps

Clickable areas in an image.

<img src="map.png" usemap="#fahimmap" alt="Map">
<map name="fahimmap">
  <area shape="rect" coords="34,44,270,350" href="https://example.com" alt="Link">
</map>

5.1 Unordered List

A list with bullet points.

<ul>
  <li>Fahim's Item 1</li>
</ul>
  • Fahim's Item 1

5.2 Ordered List

A list with numbers.

<ol>
  <li>Step by Fahim</li>
</ol>
  1. Step by Fahim

5.3 Description Lists

List with terms and descriptions.

<dl>
  <dt>HTML</dt>
  <dd>Described by Fahim</dd>
</dl>
HTML
Described by Fahim

6. Tables

Used to display tabular data.

<table border="1">
  <tr><th>Name</th></tr>
  <tr><td>Fahim</td></tr>
</table>

Fahim

8.1 <progress>

Shows progress of a task.

<progress value="70" max="100"></progress>

8.2 <meter>

Represents a scalar measurement.

<meter value="0.6"></meter>

Inline code representation.

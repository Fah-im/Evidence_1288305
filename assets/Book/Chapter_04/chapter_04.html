1. Semantic Elements

1.1 <blockquote>

Defines a section that is quoted from another source.

<blockquote>Fahim said, "Practice makes perfect."</blockquote>
Fahim said, "Practice makes perfect."

1.2 <address>

Defines contact information for the author.

<address>Written by Fahim, Dhaka, Bangladesh</address>
Written by Fahim, Dhaka, Bangladesh

1.3 <details> and <summary>

Used to create expandable content.

<details>
  <summary>More about Fahim</summary>
  <p>Fahim is learning web development.</p>
</details>
More about Fahim

Fahim is learning web development.

1.4 <figure> and <figcaption>

Used to group images with a caption.

<figure>
  <img src="./pexels-amir-1183434.jpg"  alt="Cat's Photo">
  <figcaption>cat's in 2025</figcaption>
</figure>

2. Semantic Layout Elements

Semantic tags like <header>, <nav>, <section>, <article>, <footer> define the layout clearly.

<header>Fahim's Header</header>
<nav>Navigation Links</nav>
<section>Main Section</section>
<article>Fahim's Article</article>
<footer>Footer Info</footer>
Fahim's Header
Navigation Links
Main Section
Fahim's Article
Footer Info

3. Semantic Formatting Elements

3.1 <strong>

Indicates strong importance (bold and semantic meaning).

<strong>Fahim is confident.</strong>
Fahim is confident.

3.2 <em>

Emphasizes text (italic and semantic meaning).

<em>Fahim really wants to learn!</em>
Fahim really wants to learn!

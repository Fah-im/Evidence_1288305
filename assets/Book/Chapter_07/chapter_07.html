This page demonstrates key SVG and Canvas features with code and output examples.

SVG Graphics

Rectangle

<svg width="100" height="100">
  <rect x="10" y="10" width="80" height="60" stroke="red" stroke-width="2" fill="yellow" />
</svg>

Circle

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="red" stroke-width="2" fill="yellow" />
</svg>

Line

<svg width="100" height="100">
  <line x1="0" y1="0" x2="100" y2="50" stroke="red" stroke-width="2" />
</svg>

Gradient

<svg height="100" width="200">
  <defs>
    <linearGradient id="grad" x1="0%" y1="0%" x2="100%" y2="0%">
      <stop offset="0%" stop-color="yellow" />
      <stop offset="100%" stop-color="red" />
    </linearGradient>
  </defs>
  <rect width="200" height="100" fill="url(#grad)" />
</svg>

Canvas Examples

Text

ctx.font = "20px Arial";
ctx.fillText("Fahim draws text", 10, 50);

Line

ctx.moveTo(0, 0);
ctx.lineTo(300, 100);
ctx.stroke();

Circle

ctx.arc(150, 50, 40, 0, 2 * Math.PI);
ctx.stroke();

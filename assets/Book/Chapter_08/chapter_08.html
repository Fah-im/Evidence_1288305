1. Data URIs

Data URIs let you embed small files like images directly into HTML using base64 encoding.

<img src="data:image/svg+xml;base64,..." alt="circle" />

2. Block Elements

Block elements start on a new line and take up the full width.

<div>This is a block element</div>
<p>Another block element</p>
This is a block element

Another block element

3. Inline Elements

Inline elements flow with the text and only take up as much space as needed.

<span>Inline 1</span> <span>Inline 2</span>
Inline 1 Inline 2

4. Inline-Block Elements

Inline-block elements are like inline elements, but you can set width and height.

<div class="inline-box">Box 1</div>
<div class="inline-box">Box 2</div>

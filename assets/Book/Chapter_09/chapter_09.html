1. Window Events

Window events respond to changes in the browser, like loading or resizing.

<script>
window.onload = function () {
  alert("Welcome! Page is fully loaded.");
};
</script>

2. Form Events

Triggered when users interact with form fields (focus, change, submit).

<form onsubmit="handleSubmit(event)">
  <input onchange="handleChange()" placeholder="Change me" />
  <button>Submit</button>
</form>

React to key presses: keydown, keyup, and keypress.

5. Clipboard Events

Listen for copy, cut, and paste actions.

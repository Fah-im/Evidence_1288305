1. CSS Syntax

CSS syntax consists of a selector and a declaration block.

selector {
  property: value;
}
This paragraph is blue using CSS syntax!

2. Comments in CSS

CSS comments are written inside /* ... */ and ignored by the browser.

/* This is a CSS comment */
p {
  color: green;
}
This is a green paragraph. Comment did not affect it.

3. CSS Units

Common CSS units include px, em, rem, %, vh, and vw.

.box {
  width: 50%;
  font-size: 1.5em;
}
This box is 50% wide and 1.5em in font size.

4. Shorthand Properties

Shorthand properties combine multiple properties into one.

/* Instead of writing: */
margin-top: 10px;
margin-right: 15px;
margin-bottom: 10px;
margin-left: 15px;

/* Use shorthand: */
margin: 10px 15px;
Shorthand margin applied.

5. Where Do We Put Our CSS?

You can write CSS in 3 places:

  • Inline – directly inside HTML tags.
  • Internal – inside a <style> tag in the <head>.
  • External – linked from a separate .css file.
<style>
  p { color: purple; }
</style>

<p style="color: red;">Inline style</p>

This text is red (inline style).

This text is purple (internal style).

6. User Agent Style Sheets

Browsers apply default CSS styles to elements (called user agent styles).

/* Browsers render headings bold by default */
<h2>Hello World</h2>

This heading is bold by default (user agent style).

1. Universal Selector (*)

Selects all elements on the page.

* {
  color: darkblue;
}

This paragraph is affected by the universal selector.

2. Type Selector

Targets elements by tag name.

p {
  background: #eef;
}

This <p> tag has a light blue background.

3. ID Selector

Targets a single element using its ID.

#special {
  color: green;
}

This is styled by ID selector.

4. Class Selector

Selects elements with a given class name.

.red-text {
  color: red;
}

This is red using a class selector.

5. Multiple Class Selector

Targets elements that have **both** classes.

.big.red {
  font-size: 24px;
  color: red;
}

This is big and red using multiple class selector.

6. CSS Pseudo-Class (:hover, :first-child)

Targets elements in a specific state.

button:hover {
  background-color: yellow;
}

7. CSS Pseudo-Element (::first-letter)

Targets part of an element (like first letter).

p::first-letter {
  font-size: 200%;
  color: red;
}

This paragraph has a large red first letter.

8. Combined Selector (descendant, child, sibling)

Combines selectors to match based on hierarchy.

div > span {
  color: purple;
}
This span is a direct child of div.

9. Attribute Selector

Targets elements with specific attributes.

input[type="text"] {
  border: 2px solid blue;
}

10. Selector Grouping

Applies the same style to multiple selectors.

h2, p {
  font-family: 'Georgia', serif;
}

Heading using Georgia

Paragraph using Georgia

3. CSS Outline

Outline is a line drawn outside the element border. It does not take space.

.outline-example {
  outline: 2px dashed red;
}
Outline Example

4. CSS Border

Border wraps the padding and content areas of the element.

.border-example {
  border: 3px solid green;
}
Border Example

5. CSS Padding

Padding is space between the content and border.

.padding-example {
  padding: 30px;
}
Padding Example (30px space inside the box)

6. CSS Width and Height

Defines the dimensions of the content area.

.dimension-example {
  width: 150px;
  height: 100px;
}

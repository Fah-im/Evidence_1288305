1. Color Names

CSS provides 140 named colors such as red, blue, green.

background-color: red;
This box uses the color name red.

2. HEX Colors

Hexadecimal colors are written as #RRGGBB. Each pair represents red, green, and blue values in hex.

background-color: #ff6600;
This box uses HEX color #ff6600 (orange).

3. RGB Colors

RGB stands for Red, Green, Blue. Values range from 0 to 255.

background-color: rgb(0, 128, 255);
This box uses rgb(0, 128, 255) (sky blue).

4. Grey Colors

Grey colors can be specified by using equal RGB values or greyscale names.

background-color: #888888;
This box uses HEX grey #888888. 
background-color: rgb(200, 200, 200);
This box uses light grey with rgb(200, 200, 200). 
background-color: darkgray;
This box uses named color darkgray.

1. CSS Background Properties

Control image position, repeat, size, and more using background properties.

background-image: url('image.jpg');
background-repeat: no-repeat;
background-position: right top;
background-size: 100px 100px;

2. CSS Gradient Backgrounds

Create smooth transitions between colors using linear or radial gradients.

background: linear-gradient(to right, #ff7e5f, #feb47b);

3. CSS Sprites

Combine multiple images into one file to reduce HTTP requests. Use background-position to show specific parts.

.icon-heart {
  background-image: url('sprite.png');
  background-position: -32px -32px;
}

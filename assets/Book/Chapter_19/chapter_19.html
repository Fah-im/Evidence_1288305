1. CSS Position

Controls how elements are positioned: static, relative, absolute, fixed, or sticky.

position: relative;
top: 10px;
left: 20px;
Relative Box

2. CSS Float

Moves elements to the left or right, letting text wrap around them.

float: left;
Float

This text wraps around the floated box.

3. CSS z-index

Controls the stack order of elements (higher z-index appears on top).

position: absolute;
z-index: 2;

Box 2

4. CSS Flexbox

Flexible layout system that arranges items in a row or column with control over alignment and spacing.

display: flex;
justify-content: space-between;

5. CSS Grid

Two-dimensional layout system with rows and columns.

6. CSS Alignment

Align items using Flexbox (e.g., center both vertically and horizontally).

display: flex;
justify-content: center;
align-items: center;

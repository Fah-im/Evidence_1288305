1. CSS Transitions

1.1 Basic Transitions

Transition smoothly changes properties over a duration.

background-color: #3498db;
transition: background-color 0.5s ease;

:hover {
  background-color: #2ecc71;
}
Hover Me

1.2 Transition Delay

Delay before the transition starts.

transition: background-color 0.5s ease 1s;

:hover {
  background-color: #9b59b6;
}
Hover Me (delay)

1.3 Transition Timing Function

Defines the pace of the transition.

transition: transform 0.7s cubic-bezier(0.68, -0.55, 0.27, 1.55);

:hover {
  transform: scale(1.3);
}
Hover Me (timing)

1.4 Shorthand Transition Property

Combine multiple transition properties.

transition: all 0.6s ease-in-out;

:hover {
  background-color: #1abc9c;
  transform: rotate(10deg);
}
Hover Me (shorthand)

2. CSS Animations

2.1 Two-step Keyframes

Animate between two states.

@keyframes twoStep {
  0% { background-color: #2980b9; }
  100% { background-color: #27ae60; }
}

animation-name: twoStep;
animation-duration: 2s;
animation-fill-mode: forwards;
Two-step Animation

2.2 Multiple Keyframes

Animate with several intermediate steps.

@keyframes multiStep {
  0% { background-color: #8e44ad; transform: translateX(0); }
  50% { background-color: #d35400; transform: translateX(100px); }
  100% { background-color: #16a085; transform: translateX(0); }
}

animation-iteration-count: infinite;
Multiple Keyframes

2.3 Animation Delay

Delay before animation starts.

animation-delay: 1.5s;
Delayed Animation

2.4 Animation Iteration Count

Number of times animation repeats.

animation-iteration-count: 3;
3 Iterations

2.5 Animation Direction

Normal or alternate playback direction.

animation-direction: alternate;
Alternate Direction

2.6 Animation Timing Function

Speed curve of the animation.

animation-timing-function: ease-in-out;
Ease-in-out Timing

2.7 Shorthand Animation Property

Combine multiple animation properties.

animation: multiStep 3s ease-in-out infinite alternate;
Shorthand Animation

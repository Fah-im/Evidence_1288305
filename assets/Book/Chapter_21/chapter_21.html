1. CSS Variables

CSS variables (custom properties) store reusable values.

:root {
  --main-color: #3498db;
  --padding: 10px;
}

.element {
  background-color: var(--main-color);
  padding: var(--padding);
}
Using CSS Variables

2. CSS Calculation

calc() allows you to perform calculations to determine CSS values.

.element {
  width: calc(100% - 100px);
}
Width: calc(100% - 100px)

3. CSS Counters

CSS counters let you automatically number elements using pure CSS.

.counter-list {
  counter-reset: section;
}

.counter-list li::before {
  counter-increment: section;
  content: "Step " counter(section) ": ";
}

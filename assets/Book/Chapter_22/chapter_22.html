1. Responsive Web Design (RWD)

RWD makes websites look good on all devices by adjusting layout using flexible grids and media queries.

2. RWD Viewport

The viewport meta tag helps control layout on mobile browsers.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

3. CSS Media Queries

Media queries apply different styles depending on screen size or device features.

@media (max-width: 600px) {
  .responsive-media {
    background-color: red;
  }
}
Resize the screen to see media query in action

4. RWD Grid Media

Grid layout adjusts automatically using auto-fit and minmax.

.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
}

5. Mobile-first vs Desktop-first

Mobile-first designs start with small screens and expand using min-width media queries.
Desktop-first designs start large and shrink using max-width.

Mobile-first Example

.box {
  background: lightgray;
}

@media (min-width: 768px) {
  .box {
    background: lightgreen;
  }
}
Mobile-first design

Desktop-first Example

.box {
  background: lightblue;
}

@media (max-width: 768px) {
  .box {
    background: orange;
  }
}
